# Aviation-Dashboard

This project presents an interactive data visualization dashboard built using D3.js to analyze historical aviation accident data. The dataset contains records of aviation incidents, including details such as operator names, dates, and number of fatalities, with more than 1000 entries.

The dashboard provides multiple coordinated visualizations to explore patterns and trends in aviation safety. A bar chart highlights the top operators based on total fatalities, while a line chart shows how fatalities have changed over time. Users can interact with the dashboard through features such as filtering by operator, selecting specific data points, and dynamically updating views.

The system incorporates key interaction techniques including selection, filtering, encoding through color scales, connection between multiple charts, reconfiguration of data views, tooltip-based elaboration, and data abstraction through aggregation. These features enhance usability and allow users to gain meaningful insights into accident trends and operator impact.

Overall, the dashboard demonstrates how interactive visualization can be used to transform complex aviation data into an intuitive and insightful analytical tool.
